# Lector de Imágenes

Proyecto desarrollado en Java que permite cargar y visualizar imágenes mediante una interfaz gráfica, aplicando conceptos de programación orientada a objetos y manejo de archivos.

## Tecnologías
- Java
- Java Swing
- NetBeans

## Funcionalidades
- Carga de imágenes
- Visualización de imágenes
- Selección de archivos
- Interfaz gráfica interactiva

## Cómo ejecutar
1. Clonar repositorio
2. Importar en NetBeans
3. Ejecutar Main.java

## Autor
Kevin Fajardo

## Objetivo
Aprender el manejo de imágenes, interfaces gráficas y archivos en Java.

## Estructura
- src/: código fuente
- images/: recursos e imágenes
